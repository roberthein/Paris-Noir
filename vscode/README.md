# Paris Noir

**A beautiful dark theme. Bonne nuit!**

Paris Noir is a carefully crafted dark theme for VS Code, designed for long nights, deep focus, and visual calm. Subtle contrast, soft highlights, and restrained colors reduce eye strain while keeping your code readable and expressive.

## Features
- Elegant, low-contrast dark palette  
- Excellent readability across popular programming languages  
- Thoughtful syntax highlighting without visual noise  
- Ideal for late-night coding sessions
